# TurboQuant Explained

A static GitHub Pages explainer for TurboQuant, covering:

- quasi-sparsity in LLM vectors
- random rotation / preconditioning
- PolarQuant as magnitude + direction encoding
- QJL residual sign bits
- vector database search with compressed candidate retrieval and FP32-query reranking
- KV-cache compression for long-context LLM inference

Open `index.html` directly or serve the folder with any static file server.
